# ASM-Hawk
*Nền tảng ASM tự động hóa quy trình trinh sát và xác thực rủi ro dựa trên Threat Intelligence.*


## Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React/Next.js |
| API | NestJS + Prisma |
| Core Engine | Go |
| TI Workers | Python |
| Queue | Redis + BullMQ |
| Database | PostgreSQL + ClickHouse |
